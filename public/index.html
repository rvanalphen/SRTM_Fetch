SRTM Request

Use the Inputs Below to Get Ground Elevation Based On SRTM Topography

Welcome to my Web App! Using this application you can input your desired latitude, longitude, and SRTM resolution and receive back the elevation at that location





This data is accessed through the use of OpenTopography and their API. Learn more about OpenTopography here
The data itself is from The Shuttle Radar Topography Mission:
Farr, T.G., Rosen, P.A., Caro, E., Crippen, R., Duren, R., Hensley, S., Kobrick, M., Paller, M., Rodriguez, E., Roth, L., Seal, D., Shaffer, S., Shimada, J., Umland, J., Werner, M., Oskin, M., Burbank, D., Alsdorf, D. (2007), The Shuttle Radar Topography Mission, Rev. Geophys., 45, RG2004. https://doi.org/10.1029/2005RG000183
